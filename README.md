# Jillamy DTX Warehouse SMART Floor Plan

Single-page interactive warehouse floor plan built with **React + TypeScript + Vite + Tailwind**.

## Requirements

- Node.js (LTS recommended)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- **Location format**: `ROW-AISLE-SPOT`
- **Rows**: `I, A–G`
- **Aisles**: `1–9`
- **Spots**: `1–9`
